# Termius 汉化工具

一款现代化的 Termius 汉化工具，基于 .NET 8 和 WinUI 3 构建。

## ✨ 功能特性

- 🔍 智能检测 Termius 安装路径
- 🚀 一键下载并安装汉化包
- 💾 自动备份原文件，支持随时还原
- 🎨 现代化 Win11 风格界面

## 📦 下载安装

前往 [Releases](https://github.com/FengYing1314/TermiusCN-Tool/releases) 下载最新版本。

### 系统要求

- Windows 10 (Build 19041) 或更高版本
- x64 架构

## 🚀 使用方法

1. 下载并运行 `TermiusCN-Tool.exe`
2. 工具会自动检测 Termius 安装位置
3. 选择汉化类型，点击「一键汉化」
4. 重启 Termius 即可
